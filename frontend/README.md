# Enterprise Agentic RAG Frontend

A Next.js 14 application providing a premium, glassmorphism UI for interacting with the backend agent.

## Setup

First, install the dependencies (ensure you have Node.js installed):
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
